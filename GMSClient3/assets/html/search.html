Search lets you search for landmarks.
  • How it works? - when you search for landmarks in order to customize search engine you could add to your search phrase different suffixes:
    • /p - exact phrase search,
    • /w - any phrase words search,
    • /f - fuzzy (similar) words search,
    • /a - search for address only,
    • /c - similar to /a but will not append automatically country to address. Use it if you specify country in your phrase or you search for address in different country then your phone country settings are,
    • /l - show save as layer dialog.
  • Examples
    • Restaurant - search for Restaurant according to default search method. For more details see Search Configuration section below,
    • Restaurant/l - search for Restaurant and save as custom layer,
    • Restaurant/f - fuzzy search for Restaurant,
    • New York, Wall Street/a - search for address New York, Wall Street,
    • United States, New York, Wall Street/c - search for address New York, Wall Street.
  • Search configuration
    • You could change default search method (exact phrase, phrase words or fuzzy) in settings. Go to Menu -> Settings -> Search type,
    • If you want to search for address only go to Menu -> Search location or append /a or /c to your search phrase.
